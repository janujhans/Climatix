Steps to execute the React Weather Dashboard

1. Install Node.js (LTS version) on your system.
2. Verify installation by running:
node -v
npm -v
3. Open the project folder React Weather Dashboard in Visual Studio Code.
4. Open the terminal inside the project directory.
5. Install all required dependencies mentioned in the project:
npm install
6. Ensure internet connectivity for API communication.
7. Start the React development server:
npm run dev
8. The application will compile and run using Node.js.
9. Open a browser and access:
http://localhost:3000
10. The React Weather Dashboard will display real-time weather data and forecasts successfully.
